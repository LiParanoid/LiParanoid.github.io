Java面试题（一）

发表于 2018-12-24

《Java面试题(一)》

如下代码的运行结果

这是一道关于自增变量的面试题，运行结果为：
​
i = 4；
j = 2；
k = 11;

分析如下：

首先看 i = i++ 
"=" 右边的先进行计算，赋值运算最后执行
因为"++"在右边，所以先把i的值压入操作数栈，此时计算i++，i由1变为2，
此时赋值运算还未执行，执行赋值运算时，将操作数栈中i（值为：1）的值赋给i（值为：2），这句话执行完成，i最终被赋值为1
这个过程可以看一下字节码指令
第一行：iconst_1 定义一个常量，istore_1 [i] 将常量的值赋给变量；
第二行：iload_1 [i] 将变量中的常量值压入操作数栈，iinc 1 1 ++i进行自加,istore_1 将操作数栈中的值赋值给变量i，所以i曾经为2，最终被操作数栈中的值赋成了1；
第三行：iload_1 [i] 将变量中的常量值压入操作数栈,iinc 1 1 ++i进行自加,istore_1 将操作数栈中的值赋值给变量j，所以i变为了2，j被操作数栈中的值赋成了1；
第四行：iload_1 [i]（i = 2） 将变量i中的常量值压入操作数栈，接着执行“++i”,iinc 1 1,i进行自加,然后这个值并未被赋与其它变量，再次读入i的值 iload_1 [i]（i = 3），将变量i中的常量值压入操作数栈，再执行“i++”，读入i的值 iload_1 [i]（i = 3），iinc 1 1,i进行自加，最终 i= 3，参与运算的值分别为 2 + 3 * 3 
k最终为：11
